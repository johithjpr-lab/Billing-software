### Quick Start
1. Open **Stock Manager**
2. Add product details (Code, Name, Rate, Unit, etc.)
3. Click **"Generate QR & Scan#"** button
4. A modal appears with:
   - QR Code (280x280px)
   - Unique Scan Number (SC-[timestamp]-[random])
   - Product Details
5. **Print**, **Download**, or **Copy** the QR code
6. Click **"Add / Update"** to save product to stock
